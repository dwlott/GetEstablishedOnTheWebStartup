# Local WordPress Build Is Live

The **GetEstablishedOnTheWeb.com** product repository now syncs Markdown page
source to a local **WordPress + Genesis + Altitude** staging site on WAMP.

## What this proves

- Public page copy lives in **`Content/Website/Pages/*.md`** and syncs via CLI scripts.
- **Capability posts**, **repository offer hubs**, and **primary navigation** rebuild from manifests — not hand-edited in wp-admin.
- Inner-page styling, link behavior, and visitor-first nav (**Get Listed**, **Use AI**, **Resources**) are testable before Bluehost production.

## Where to look

- Sync scripts: `Workspace/LocalWordPressBuild/`
- Navigation manifest: `nav-menu-manifest.php`
- Local verify URL pattern: `http://localhost/getestablishedontheweb/`
