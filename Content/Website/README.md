# Website

## Purpose

This folder holds **example** website Markdown for you to adapt — not a finished
public site.

## What Belongs Here

- Page drafts under `Pages/` that you review and customize for your business.
- Future content planning artifacts specific to your site.

## What Does Not Belong Here

- Production build scripts or WordPress automation (product-repo scope).
- Generated site output or deployment credentials.
- General project plans that belong under `Plans/`.

## Important Folders

- `Pages/` — example plain Markdown page drafts from the web-presence starter pack.

## Agent Notes

Treat all pages as **examples** until **ContentReview** and the owner confirm
claims and readiness for publication.
