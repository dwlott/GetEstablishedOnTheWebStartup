# Pages

## Purpose

This folder holds plain Markdown source drafts for future website pages.

## What Belongs Here

- Framework-neutral page drafts.
- Page-specific notes that help future content refinement.
- Drafts that can later be adapted into a website build.

## What Does Not Belong Here

- HTML, CSS, JavaScript, or framework-specific files unless a build approach has been chosen.
- Generated website output.
- General project planning files.
- Automation scripts.

## Important Files

- `Index.md` - overview of the page drafts and drafting rules.
- `Home.md` - homepage source draft.
- `StartHere.md` - beginner entry page draft.
- `HowItWorks.md` - method overview draft.
- `AIWorkflow.md` - AI workflow page draft.
- `GitHubWorkflow.md` - GitHub workflow page draft.
- `Offers.md` - offer direction draft.
- `About.md` - project background draft.

## Agent Notes

Preserve these files as source content. Keep edits practical, credible, and free of unsupported hype.
