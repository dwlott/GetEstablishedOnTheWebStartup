# Get Established On The Web

**GetEstablishedOnTheWebStartup** is the **primary repository package** for this site — the web-presence platform with the core GetEstablished method plus the **web Capability group** for listings, page content, and WordPress workflows.

Use it when your project is specifically about **getting established on the web**.

## Download

**[Download GetEstablishedOnTheWebStartup (ZIP)](download:GetEstablishedOnTheWebStartup.zip)** (~450 KB)

1. Download the ZIP and unzip it to a stable folder (not `Downloads`).
2. Open the folder in your favorite AI platform.
3. Run **Begin Quick Startup from AGENTS.md**.

Not sure yet, or do not need the WordPress Capabilities? **[GetEstablishedStartup](GetEstablished.md)** is the lighter core option you can try first.

## Capabilities

**[Capabilities](Capabilities.md)** are the interesting part — packaged workflows your agent reads and follows. GEOTW includes the full core platform **plus** the web Capability group.

Standouts in this package:

- **[Situational Awareness](capability:ge-cap-situational-awareness)** — orient every agent pass before work begins.
- **[Getting Started](capability:ge-cap-getting-started)** — **Begin Quick Startup** writes your Owner Goals and first tasks.
- **[Web Presence Node](capability:ge-cap-web-presence-node)** — plan and track one listing or profile at a time across Google, Bing, Facebook, Nextdoor, Foursquare, BBB, Yelp, Apple Maps, and Yellow Pages.
- **[One Page Website](capability:ge-cap-one-page-website)** — scaffold and grow site content from Markdown.
- **[Content Review](capability:ge-cap-content-review)** — review drafts before they go public.
- **[Chat Memory Capture](capability:ge-cap-chat-memory-capture)** — turn useful chat into durable repository files.

The **Capability posts below** list everything in GetEstablishedOnTheWebStartup. Browse the **[full Capability catalog](Capabilities.md)** for every public module on this site.

## GetEstablished vs Get Established On The Web

Both repositories share the same harmonized core. **GetEstablishedOnTheWebStartup** stays in front because it adds the web modules listed above and room to grow into listings and WordPress workflows.

Need the universal method only? **[GetEstablished](GetEstablished.md)** is the lighter core platform.

## Practical listing work on this site

**[Get Listed](GetListed.md)** walks through Google, Yelp, Apple Maps, and Yellow Pages with official links.

Your repository holds the planning files so an agent can help with listing steps, page drafts, and next actions — organized and reviewable on your computer.
