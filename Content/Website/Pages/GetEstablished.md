# GetEstablished

**GetEstablishedStartup** is the **lighter core repository** — a modular, fully indexed GetEstablished workspace without the web and WordPress Capability group.

Use it when you want to try the method first, or when your project does not need the full **[Get Established On The Web](GetEstablishedOnTheWeb.md)** package.

## Download

**[Download GetEstablishedStartup (ZIP)](download:GetEstablishedStartup.zip)** (~2 MB)

1. Download the ZIP and unzip it to a stable folder (not `Downloads`).
2. Open the folder in your favorite AI platform.
3. Run **Begin Quick Startup**.

Plain-language guide: **[Use AI](UseAI.md)**.

## Capabilities

**[Capabilities](Capabilities.md)** are the interesting part — packaged workflows your agent reads and follows. Each one has Rules, a Prompt, and an index so the next pass starts faster than the last.

Standouts in the core platform:

- **[Situational Awareness](capability:ge-cap-situational-awareness)** — orient every agent pass before work begins.
- **[Getting Started](capability:ge-cap-getting-started)** — **Begin Quick Startup** writes your Owner Goals and first tasks.
- **[Assisted Agentic Workflow](capability:ge-cap-assisted-agentic-workflow)** — structured handoffs between you and the agent.
- **[GitHub Workflow](capability:ge-cap-github-workflow)** — connect your repository when you are ready for backup and history.
- **[Content Review](capability:ge-cap-content-review)** — review drafts before they become public.

The **Capability posts below** list everything in GetEstablishedStartup. Browse the **[full Capability catalog](Capabilities.md)** for every public module on this site.

## How It Relates To Get Established On The Web

**GetEstablishedOnTheWebStartup** is the main package for this site. It includes the same harmonized core plus web-presence modules for listings, page content, and WordPress workflows.

Choose **GetEstablishedStartup** when you do not want the web package yet. Building for the web? See **[Get Established On The Web](GetEstablishedOnTheWeb.md)** and pair directory work with **[Get Listed](GetListed.md)**.
