# GoogleBusinessProfile Capability

- **Version:** 1
- **Tier:** Archetype
- **Status:** Planned
- **Purpose:** Guide a repository owner through preparing, claiming, creating, or
  reviewing a Google Business Profile using official references and conservative
  claims boundaries.
- **Inputs:** Owner-approved business facts; current profile status if known;
  official Google help links; source content from GEOTW.
- **Outputs:** Repository Markdown checklist, review notes, or next-step plan.
  No external profile edits until the owner explicitly approves that pass.
- **WhenToUse:** A web-presence repository needs Google Business Profile basics,
  setup planning, or profile-readiness review.
