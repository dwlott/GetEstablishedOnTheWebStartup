# Reference Review Prompt

Use when Quick Startup or an early pass added **Suggested** rows to
`Workspace/ValuableReferences.md`. Keep the review short — five minutes is enough.

```text
# Owner pass: Review valuable references

Read Workspace/ValuableReferences.md.

For each row with Confidence: Suggested:
1. Open or search the URL (or use the search term provided).
2. Decide: confirm, edit, demote to Deprecated, or delete the row.
3. Update Confidence to Owner-confirmed when you approve a row.
4. Add Review notes when a link is wrong, outdated, or junk.

Do not block other work on finishing every row — review what matters for the
current next task first.

When done, name one next task that uses the confirmed references (if any).
```

## When To Offer This Review

- End of Quick Startup (optional): "I added N suggested references — skim now or after your first task?"
- Before first work that **depends** on an external canonical source (exam outline, official docs, pricing page).
- When Suggested row count grows without owner review.
