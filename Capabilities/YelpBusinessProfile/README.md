# YelpBusinessProfile Capability

- **Version:** 1
- **Tier:** Archetype
- **Status:** Planned
- **Purpose:** Guide a repository owner through preparing, claiming, creating, or
  reviewing a Yelp Business Page using official references and conservative
  review-policy boundaries.
- **Inputs:** Owner-approved business facts; current page status if known;
  official Yelp for Business links; source content from GEOTW.
- **Outputs:** Repository Markdown checklist, review notes, or next-step plan.
  No external Yelp page edits until the owner explicitly approves that pass.
- **WhenToUse:** A web-presence repository needs Yelp Business Page basics, setup
  planning, or profile-readiness review.
