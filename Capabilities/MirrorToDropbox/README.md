# MirrorToDropbox Capability (Stub Alias)

**Mirror refresh to a Dropbox path** is owned by the **platform mirror Capability**:

| Platform | Route |
| --- | --- |
| Windows | [../MirrorToWindows/WorkflowIndex.md](../MirrorToWindows/WorkflowIndex.md) |
| macOS | [../MirrorToMac/WorkflowIndex.md](../MirrorToMac/WorkflowIndex.md) |

Dropbox Desktop sync folders are **mirror target paths** on Windows — not a
separate apply Capability.

## Use Instead

| Task | Start here |
| --- | --- |
| Mirror / refresh to Dropbox | [../MirrorToWindows/WorkflowIndex.md](../MirrorToWindows/WorkflowIndex.md) |

`Workspace/OwnerPreferences.md`.
